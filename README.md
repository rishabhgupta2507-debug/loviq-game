# Love Riddles Game 💕 — Replit Setup

## How to Run on Replit

1. Upload all these files to your Replit project
2. Click the **Run** button
3. The game opens in the Replit preview window

## How to Play Multiplayer

1. **Player 1** opens the game → enters name → clicks **"Create Game Room"**
2. A **6-character room code** appears (e.g. `AB12CD`)
3. **Player 1** shares the code or clicks **"Copy Link"**
4. **Player 2** opens the same Replit URL → enters name → pastes the code → clicks **"Join Game"**
5. Player 1 clicks **"Start Game"**
6. Both players play together in real-time!

## Features
- Real-time multiplayer via Socket.io
- 3 Levels: Easy (10 questions) → Spicy (10) → Wild (10)
- Turn-by-turn alternating gameplay
- 45 second timer per question
- In-game chat with emojis (disappears after game)
- Date night ideas (Indian themed)
- Calendar scheduling & reminders
- Share code + share link

## Tech Stack
- Backend: Node.js + Express + Socket.io
- Frontend: Vanilla HTML/CSS/JS
- No database needed!
